# Amigos_Unite_App
Vite React/Typescript front end to Rails back end using Axios and HTTP Cookies with embedded JWT token with CSRF Protection.

Amigos Unite is a platfrom where amigos and create an acount list events which when listed will be sent to all other members within a ten mile radius of hte event location. When an amigo creates an even, that amigo becomes coordinators for thart event. Other amigos can then volunteer to be assistant coordinators to help manage the event.

Amigos can upload an avatar, creates details about themselves, provide locaton iunformation.

When an amigo creates an even they also must supply locxation and contact inbformation for that event.

When an amigo registers for an event they will be added to the list of attendees so that reminders can be sent in advance of of the time of hte event.

An amigo can edit their personal information any time and upload an avatar in .svg, .jpeg, .png formats.
